# 数据模拟规范

## 1. 目标

数据模拟用于：

* 开发与调试
* 稳定测试
* 提前验证指标计算与状态机行为

---

## 2. Fake 数据源必须支持的配置

* 采样频率（Hz）
* 噪声强度
* 最大力范围
* 力曲线模式

  * 线性上升
  * 阶梯
  * 随机游走

---

## 3. 可复现模式

必须支持：

* 固定随机种子
* 同一时间基准

禁止：

* timer 与 stream 使用不同时间源

---

## 4. 时间基准规范

* 所有模拟数据时间来自同一 clock abstraction
* provider 与 service 不得各自维护 timer

---

## 5. 生命周期规范

* start 后开始产出数据
* pause 停止产出但不销毁
* dispose 后立即停止所有 stream
